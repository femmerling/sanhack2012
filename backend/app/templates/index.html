Welcome!


EmeraldBox is a boilerplate framework for developing python web applications with database access. The framework is based on Flask, a python microframework based on werkzeug, jinja2 and good intentions.

Your server is currently running. However you only have basic functions to the site.

Feel free to modify the contents and add stuffs to complete it.

Please register issues at our GitHub issue tracker and we'll deal with it as soon as we have time.
